# **App Name**: Ultimutt Pet Resort Website

## Core Features:

- Service Showcase: Display boarding, daycare, grooming, and add-on services with descriptions, pricing, and booking options.
- Booking Request Form: Allow pet parents to submit booking requests with pet and owner details.
- Image Gallery: Showcase the facility and happy pets with a filterable masonry-style gallery.
- Interactive Chatbot: An AI tool that can answer boarding queries, provide daycare timings, explain rates, list packing essentials, and clarify vaccination needs.

## Style Guidelines:

- Background color: Light pink (#FDE8EB), evoking warmth and gentleness.
- Primary color: Medium-soft pink (#E6A4B2) for a calming, trustworthy feel.
- Accent color: Lightest pastel pink (#F7D6DA) used on hovers to create contrast while retaining the site's soothing feel.
- Headline font: 'Playfair' serif for an elegant feel.
- Body font: 'Inter' sans-serif for clean readability.
- Subtle animations on scroll, button hover, and card hover to add a modern touch.
- Consistent use of border radius (0.75rem) and a centered container layout (max-width 1400px).
